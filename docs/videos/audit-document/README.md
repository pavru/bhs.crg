# Видеоинструкция: аудит документа (пользовательский режим)

Как пользователь сам «лечит» свой документ, если его данные разошлись с текущей схемой типа
(осиротевшие поля) — без обращения к администратору.

- **`instruction.mp4`** — немой скринкаст (~28 сек, H.264). `instruction.webm` — исходник.
- **`voiceover.md`** — текст озвучки с таймкодами под видео.

Сценарий: открыть документ → вкладка «Генерация» → «Аудит документа» → удалить/переименовать
осиротевшее поле → расхождений нет.

> Скринкаст немой; для готового ролика начитайте текст из `voiceover.md` (голосом или TTS) по таймкодам.
